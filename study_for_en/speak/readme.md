## 口语

### 常见误区

- 情景式口语 // 很多教材都是这种，但实际用处不大
  - 对话式口语 // 这个常用
  - 国内常见的口语场景: 面试、外企、考试
- 生存式口语 // 教材很多都是这个，用户不大
  - 实质性话题 // 更为实用
    - 谈工作
    - 聊想法
    - 讲故事
- 单句教学多，段落教学少
- 忽略口音教学
  - 口音是一个人的门面
- 只教口语，不教听力
  - 听力比口语难很多
- 语言教学多，素质教学少
- 重词汇，轻术语
  - 闯红灯
  - 高跟鞋
  - 日光灯
  - 无名指
  - 勾股定理
  - 大便、小便、放屁
  - 杂交水稻
  - 里氏7.0级
- 盲目迷信外教，轻视教学技术
  - 懂英语!=会教英语
