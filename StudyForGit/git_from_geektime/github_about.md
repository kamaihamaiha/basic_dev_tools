### 快速找到感兴趣的开源项目

- 按照创建时间搜索: ``created:<2024-07-31``
- 直接在搜索框搜索关键字的效果: 搜索仓库的名称和描述
- 搜索 readme 中的关键字: ``git 最好 学习 资料 in:readme``
- 加限制条件，超过1000 star 的仓库: ``git 最好 学习 资料 in:readme stars:>1000``
