## VS Code 使用教程

[教程视频](https://www.bilibili.com/video/BV1ty4y1S7mC?p=2&spm_id_from=333.880.my_history.page.click&vd_source=82b7ac2fbd7ece380f983e2c23199d99)

- 显示 「Get Started」页面
  - 打开 「Command Palette」
  - 然后搜索：
    - Help: get started （如果是编辑器语言是英文）
    - Welcom 或者 欢迎使用（如果编辑器语言是中文）
- 快捷键查询
  - 打开 「Command Palette」
  - 然后搜索：shortcuts
  - 选中后，会打开一个快捷键的网页
